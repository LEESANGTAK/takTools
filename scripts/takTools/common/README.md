# common - 도메인 도구 모듈

이 패키지에는 Maya 작업 도메인(리깅, 모델링, 파이프라인 등)에서
공통으로 사용되는 **고수준 도구 함수와 UI**가 포함됩니다.

## 모듈 구조

| 모듈 | 역할 |
|------|------|
| `tak_misc.py` | 기타 유틸리티 (하위 호환 re-export 허브) |
| `tak_skin.py` | 스킨 웨이트, 디포머 관련 |
| `tak_display.py` | 뷰포트 디스플레이, 색상 관련 |
| `tak_mirror.py` | 미러 조인트/오브젝트/컨트롤러 |
| `tak_lib.py` | 공통 헬퍼 함수 라이브러리 |
| `curves.py` | 커브 CV 데이터 |
| `enums.py` | 열거형 상수 |
| `iconMaker.py` | 아이콘 생성 도구 |

## `utils/` 패키지와의 차이점

- **`common/`**: UI가 포함된 고수준 도구, 복합 작업 수행
- **`utils/`**: UI가 없는 순수 유틸리티 함수, Maya API 래퍼, 재사용 가능한 저수준 함수
